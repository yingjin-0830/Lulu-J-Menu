# 🍜 噜噜金私厨

> 活着挺好，吃着更好 ✨ · [English](./README.md)

一个专为家庭设计的私人数字菜谱，像餐厅点单 App 一样好用。

---

## ✨ 功能

- 🃏 卡片 / 列表双视图切换
- 🔍 按菜名、食材搜索
- 📂 分类筛选（荤菜、素菜、汤类…）
- 🖼️ 支持上传菜品图片
- ⭐ 收藏喜爱的菜
- 🛒 生成今日菜单 & 购物清单
- 📅 预约菜单（一周餐食规划）
- 🌐 中英文双语切换
- 📤 一键导出分享版，发给家人一起用

---

## 🚀 使用方法

**在线访问：** 👉 [yingjin-0830.github.io/Kitchen-Menu](https://yingjin-0830.github.io/Kitchen-Menu)

或下载 `index.html`，在任意浏览器中打开即可，无需安装任何依赖，数据存储在本地浏览器中。
