# 🍜 Lulu Jin's Kitchen

> Good food, good life ✨ · [中文版](./README.zh-CN.md)

A personal digital recipe book designed for home use — feels just like ordering at a restaurant.

---

## ✨ Features

- 🃏 Card & list view toggle
- 🔍 Search by dish name or ingredient
- 📂 Category filtering
- 🖼️ Upload dish photos
- ⭐ Save favorites
- 🛒 Today's menu & auto-generated shopping list
- 📅 Weekly meal planner
- 🌐 Chinese / English toggle
- 📤 One-click export to share with family

---

## 🚀 Usage

**Live demo:** 👉 [yingjin-0830.github.io/Kitchen-Menu](https://yingjin-0830.github.io/Kitchen-Menu)

Download `index.html` and open in any browser — no installation or dependencies needed. All data is stored locally in your browser.

Plain HTML / CSS / JavaScript — single file, zero dependencies.
